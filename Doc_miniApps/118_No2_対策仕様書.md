## 📋 問題の概要

### 発生している問題

**iOS 18.6.2以降で登録QRをスキャンすると、Safari（外部ブラウザ）で開かれてしまい、LIFFアプリが正常に動作しない**

- 該当ユーザー: 3人（iOS 18ユーザー）
- 症状: LIFF初期化を53秒繰り返す → タイムアウト
- 影響範囲: iOS 18.6.2以降のみ

### 基本方針

**✅ 既存ユーザーに一切影響を与えない**

1. **正常動作しているユーザーには何も表示しない**
   - LINEアプリ内で開いているユーザー → 通常通り動作
   - iOS 17以前のユーザー → 通常通り動作
   - Androidユーザー → 通常通り動作

2. **問題が発生しているユーザーにのみ誘導UIを表示**
   - 外部ブラウザで開いているユーザー → 誘導画面を表示
   - LINEアプリへの誘導ボタンを提供

3. **既存のLIFF初期化ロジックは変更しない**
   - 現在のコードは適切に実装されている
   - 性急な変更によるリスクを回避

## 🔧 実装仕様

### 1. 外部ブラウザ判定ロジック

#### 判定タイミング

**LIFF初期化直後に判定**

```
ページ読み込み
  ↓
LIFF初期化開始 (liff.init())
  ↓
初期化成功
  ↓
liff.isInClient() で判定 ← ここで判定
  ↓
true: LINEアプリ内 → 通常通り動作 ✅
false: 外部ブラウザ → 誘導画面を表示 ⚠️
```

#### 判定条件

```typescript
// ✅ 正常（通常通り動作）
if (liff.isInClient() === true) {
  // LINEアプリ内で開いている
  // 何もせず、通常のアプリを表示
}

// ⚠️ 問題あり（誘導画面を表示）
if (liff.isInClient() === false) {
  // 外部ブラウザで開いている
  // 誘導画面を表示
}
```

### 3. 実装場所

#### app/layout.tsx に実装

**理由**:
- すべてのページで共通の処理
- 最初に実行される
- 既存のページコンポーネントに影響を与えない

#### 実装コード（概要）

```typescript
"use client";

import { useEffect, useState } from "react";
import liff from "@line/liff";

export default function RootLayout({
  children
}: {
  children: React.ReactNode
}) {
  const [isExternalBrowser, setIsExternalBrowser] = useState(false);
  const [isLiffReady, setIsLiffReady] = useState(false);

  useEffect(() => {
    const initLiff = async () => {
      try {
        await liff.init({ liffId: process.env.NEXT_PUBLIC_LIFF_ID! });
        setIsLiffReady(true);

        // LIFF環境で動作しているか確認
        if (!liff.isInClient()) {
          // 外部ブラウザで開いている
          setIsExternalBrowser(true);
          console.warn("⚠️ 外部ブラウザで開かれています");
        } else {
          // LINEアプリ内で開いている（正常）
          console.log("✅ LINEアプリ内で開かれています");
        }
      } catch (error) {
        console.error("❌ LIFF初期化エラー:", error);
      }
    };

    initLiff();
  }, []);

  // 外部ブラウザの場合、誘導画面を表示
  if (isLiffReady && isExternalBrowser) {
    return <ExternalBrowserWarning />;
  }

  // 通常のアプリを表示（既存の動作）
  return children;
}
```

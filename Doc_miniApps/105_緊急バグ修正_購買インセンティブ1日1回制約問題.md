## 📋 問題の概要

### 報告された問題

1. **15スタンプQRで10ポイントしか付与されない**
   - ユーザー報告: 優良患者用15スタンプQRをスキャンしても10ポイントしか増えない

2. **購買インセンティブQRが1回しかスキャンできない**
   - 仕様: 商品購入時のQRコード（5スタンプ）は1日何度でもスキャン可能
   - 実際: 1日1回しかスキャンできない（2回目以降エラー）

### 根本原因

**[app/api/stamps/route.ts](../app/api/stamps/route.ts)** の重複チェック（Lines 46-78）に問題があります。

```typescript
// 🔴 問題のあるコード
// 重複チェック: 同日同QRの登録済みチェック
const today = new Date().toISOString().split("T")[0];
const { data: existing, error: checkError } = await supabase
  .from("stamp_history")
  .select("id")
  .eq("user_id", userId)
  .eq("qr_code_id", qrCodeId)  // ← QRコードIDで重複チェック
  .gte("visit_date", `${today}T00:00:00`)
  .lt("visit_date", `${today}T23:59:59`)
  .maybeSingle();

if (existing) {
  return NextResponse.json(
    {
      success: false,
      message: "本日すでにスタンプを取得済みです",  // ← 購買インセンティブも拒否される
      error: "Duplicate stamp",
    },
    { status: 400 }
  );
}
```

**問題点:**
- QRコードペイロードをパースして `type: "purchase"` かどうかを確認していない
- すべてのQRコードに1日1回制限が適用される
- 購買インセンティブが無制限スキャンできない

## 🛠️ 修正方針

### 修正内容

1. **QRコードペイロードのパース**
   - `qrCodeId` をJSONパースして `type` を取得
   - パースエラー時はデフォルトで `"regular"` とみなす

2. **購買インセンティブの除外**
   - `type === "purchase"` の場合、重複チェックをスキップ
   - 重複チェックは `type !== "purchase"` の場合のみ実行

3. **stamp_method の正しい設定**
   - `type === "purchase"` → `stamp_method: "purchase_incentive"`
   - それ以外 → `stamp_method: "qr"`（QRスキャン一般）

### 修正コード

```typescript
// QRコードからスタンプ数とタイプを解析
let stampAmount = STAMP_AMOUNTS.REGULAR_VISIT; // デフォルト: +10個
let qrType: 'regular' | 'premium' | 'purchase' = 'regular';

try {
  const qrPayload = JSON.parse(qrCodeId);
  if (qrPayload.stamps && typeof qrPayload.stamps === 'number') {
    stampAmount = qrPayload.stamps;
  }
  if (qrPayload.type && ['regular', 'premium', 'purchase'].includes(qrPayload.type)) {
    qrType = qrPayload.type;
  }
  console.log(`📱 [Stamps API] QRコードから読み取り: ${stampAmount}個, type: ${qrType}`);
} catch {
  // JSONパースエラー = 通常のQRコードID、デフォルト値を使用
}

// 購買インセンティブ以外は1日1回制限チェック
if (qrType !== 'purchase') {
  // 既存の重複チェックロジック
  const today = new Date().toISOString().split("T")[0];
  const { data: existing, error: checkError } = await supabase
    .from("stamp_history")
    .select("id")
    .eq("user_id", userId)
    .eq("qr_code_id", qrCodeId)
    .gte("visit_date", `${today}T00:00:00`)
    .lt("visit_date", `${today}T23:59:59`)
    .maybeSingle();

  if (existing) {
    return NextResponse.json(
      {
        success: false,
        message: "本日すでにスタンプを取得済みです",
        error: "Duplicate stamp",
      },
      { status: 400 }
    );
  }
}

// stamp_historyに新規レコードを挿入
const { data: stampData, error: insertError } = await supabase
  .from("stamp_history")
  .insert({
    user_id: userId,
    visit_date: new Date().toISOString(),
    stamp_number: nextStampNumber,
    stamp_method: qrType === 'purchase' ? 'purchase_incentive' : 'qr',
    qr_code_id: qrCodeId,
    amount: stampAmount,
    notes: qrType === 'purchase'
      ? "購買インセンティブ（ホームボタン）"
      : `ホームボタンスキャン (${qrType})`,
  })
  .select()
  .single();
```

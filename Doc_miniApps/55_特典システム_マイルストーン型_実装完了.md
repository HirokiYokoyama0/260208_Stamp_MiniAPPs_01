# 特典システム - マイルストーン型 実装完了

**作成日:** 2026-03-27
**ステータス:** ✅ 実装完了

---

## 📢 概要

特典システムを**マイルストーン自動付与型**に変更しました。

- **10, 20, 30...** → 歯ブラシ 1本
- **50, 100, 150...** → POIC殺菌剤（初回は本体込、2回目以降は補充用）
- **300, 450, 600...** → 選べるメニュー割引（有効期限5ヶ月）

### 優先度ルール

複数のマイルストーンが重なる場合、優先度の高い特典のみ付与：

- **優先度:** 選べるメニュー割引 > POIC > 歯ブラシ
- **例:** 50スタンプ到達 → POICのみ（歯ブラシはスキップ）

---

## 💾 データベース

### テーブル構成

```
milestone_rewards     特典マスター（3種類）
reward_exchanges      交換履歴（カラム追加: milestone_reached, is_milestone_based, valid_until, is_first_time）
milestone_history     到達履歴
```

### 重要な変更

1. **外部キー制約削除**: `reward_exchanges.reward_id` の外部キー制約を削除
   - 理由: `rewards` と `milestone_rewards` の両方を参照できるようにするため

2. **新カラム**:
   - `milestone_reached`: 何スタンプ目の交換か（10, 50, 300...）
   - `is_milestone_based`: 新仕様かどうか（true）
   - `valid_until`: 有効期限（自費メニュー用）
   - `is_first_time`: 初回特典か（POIC用）

## 📊 管理ダッシュボード向け情報

### 交換履歴の取得方法

**❌ 間違い**:
```javascript
// rewards テーブルとJOINしようとする
select('*, reward:rewards(name)')  // エラー: 外部キー制約がない
```

**✅ 正しい方法**:
```javascript
// Step 1: reward_exchanges を取得
const { data: exchanges } = await supabase
  .from('reward_exchanges')
  .select('*')
  .eq('is_milestone_based', true)
  .order('exchanged_at', { ascending: false });

// Step 2: milestone_rewards を取得
const { data: rewards } = await supabase
  .from('milestone_rewards')
  .select('*');

// Step 3: 手動でJOIN
const exchangesWithRewards = exchanges.map(ex => {
  const reward = rewards.find(r => r.id === ex.reward_id);
  return {
    ...ex,
    reward_name: reward?.name || '不明',
    reward_type: reward?.reward_type
  };
});
```

### 表示すべきカラム

- `milestone_reached`: 何スタンプ目の交換か（重要）
- `is_milestone_based`: true で新仕様
- `valid_until`: 有効期限
- `is_first_time`: 初回特典か

### 原因分析

管理ダッシュボードで交換履歴が表示されない原因：

1. **rewards テーブルと JOIN しようとしている**
   - 外部キー制約がないため失敗
   - milestone_rewards を使う必要がある

2. **is_milestone_based を確認していない**
   - 新しいデータは is_milestone_based = true

3. **milestone_reached を取得していない**
   - 何スタンプ目の交換かがわからない

4. **Supabaseの自動JOINを使おうとしている**
   - 外部キー制約がないため動作しない
   - 手動JOINが必要

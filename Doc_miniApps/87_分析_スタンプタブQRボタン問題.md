# 分析：スタンプタブのQRスキャンボタン問題

**分析日**: 2026-03-28
**報告者**: クライアント
**問題**: スタンプタブの「来院スタンプを読み取る」ボタンで購買インセンティブQRを読むと1日1回制限が効いてしまう

---

## 🔍 問題の詳細

### 症状

**会員証タブ（トップページ）**: ✅ OK
- 購買インセンティブQRは何度でもスキャン可能

**スタンプタブ**: ❌ NG
- 購買インセンティブQRも1日1回制限が効いてしまう
- 「本日は既にQRコードでスタンプを受け取っています」エラー

---

```typescript
// 1日1回制限チェック（購買インセンティブは対象外）
if (type !== "purchase") {
  // 日本時間（JST = UTC+9）で当日の範囲を計算
  const nowJST = new Date(new Date().getTime() + 9 * 60 * 60 * 1000);
  const todayJST = nowJST.toISOString().split("T")[0];
  const startOfDayUTC = `${todayJST}T00:00:00+09:00`;
  const endOfDayUTC = `${todayJST}T23:59:59.999+09:00`;

  const { data: todayQrRecords } = await supabase
    .from("stamp_history")
    .select("id, stamp_method, notes")
    .eq("user_id", userId)
    .eq("stamp_method", "qr")
    .gte("visit_date", startOfDayUTC)
    .lt("visit_date", endOfDayUTC);

  if (todayQrRecords && todayQrRecords.length > 0) {
    return NextResponse.json({
      success: false,
      message: "本日は既にQRコードでスタンプを受け取っています",
    }, { status: 409 });
  }
}
```

## 💡 UX的な検討

### 現状の「来院スタンプを読み取る」ボタン配置

| タブ | ボタン配置 | 問題点 |
|-----|----------|--------|
| 会員証（トップ） | ✅ あり | なし（メインの動線） |
| スタンプ | ❓ あり | 重複、混乱の原因 |
| ケア記録 | - | - |
| 特典 | - | - |
| お知らせ | - | - |

### 問題点

1. **機能の重複**
   - 会員証とスタンプの両方に同じボタン
   - ユーザーがどちらを使えばいいか迷う

2. **動線の混乱**
   - 「来院スタンプを読み取る」という名前なのに、スタンプタブにある
   - 本来は会員証タブがメインの動線

3. **異なるAPI**
   - 会員証: `/api/stamps/scan`（新しい、機能豊富）
   - スタンプ: `/api/stamps`（古い、シンプル）
   - 同じボタンなのに裏側が違う → バグの温床

### 推奨するUX改善

#### 案1: スタンプタブのQRボタンを削除（最推奨）

**メリット:**

- ✅ 保守コストが減る

**理由:**

## 🎯 まとめ

### 問題

- スタンプタブのQRボタンが `/api/stamps` を使用
- このAPIは購買インセンティブも1日1回制限対象
- 会員証タブの `/api/stamps/scan` とは仕様が異なる

### 推奨する対応

**優先度1（必須）**: UX改善
- ✅ スタンプタブのQRボタンを削除
- ✅ 会員証タブへのリンクに変更
- 理由: ボタンの重複を解消、ユーザー体験の改善

**優先度2（任意）**: API修正
- `/api/stamps` を修正して購買インセンティブを対象外に
- 理由: 既存コードとの互換性維持

### 理由

1. **UXの問題**
   - 同じ機能のボタンが2箇所にある
   - ユーザーがどちらを使えばいいか迷う

2. **保守性の問題**
   - 2つのAPIで同じような処理
   - バグが発生しやすい

3. **動線の問題**
   - スタンプタブは「履歴を見る」場所
   - スタンプを取得するのは会員証タブが自然
